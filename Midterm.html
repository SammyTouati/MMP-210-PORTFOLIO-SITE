MidTerm Project


Members:  Sammy Touati, Angel Ferrera

Components Used: HC-SR04 Distance Sensor, Arduino Genuino Uno, USB-AB Cable, Connecting Wires, and a Bread Board.

Link to Project:  C L I C K   H E R E

I teamed up with a friend, Angel Ferrera for the MidTerm Project. The purpose of this project was to build an “object”, using the Arduino Uno, and cables (other items) that receives the distance of an object in it’s direction, and translates that information into sound. The closer an object would be to the distance sensor, the higher pitch the sound will be, and the farther an object is from the sensor, the lower the sound would be. A challenge we had to deal with, was changing the pitch of the specific sound, and getting the distance sensor to respond to the object in front of it at first.
